## How to use it?

1. Open link https://mileni67.github.io/kalkulatorHP/
2. Enter the required amount in the line "Kwota w PLN*:"
3. Select the desired currency in the line "waluta:"
4. Click "Przelicz!"
5. At the bottom of the page will be the amount
